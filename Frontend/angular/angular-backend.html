Communication with the Backend - $http


angular.module('myFirst', [])
    .factory('books', function($http){
        return {
            getOne: function (bookId) {
                return $http.get('books/' + bookId)
                    .then(function (response) {
                        return response.data;
                    });
            }
        };
    });

$http provides...

...a generic method $http(config),

as well as shortcut methods:

  • $http.get(url, [config])
  • $http.head(url, [config])
  • $http.post(url, data, [config])
  • $http.put(url, data, [config])
  • $http.delete(url, [config])
  • $http.jsonp(url, [config])
  • $http.patch(url, data, [config])

$httpBackendProvider allows to...

...set default headers for every type of request

...define interceptors for cross-cutting features

...enable XSRF protection
