Tags

Tags that are used too often

  • TABLE, TR, TD
  • DIV,SPAN
  • BR

Source of page should reflect semantic of page.

TABLE should be used where tabular data is presented... but not to achieve tabular layout.

DIV and SPAN span have no meaning ... but sometimes must be use just to distinguish small portion of HTML (for styling).

Usage of BR for layout... is a crime.
